# SkillHub Portfolio

SkillHub Portfolio is my professional personal portfolio website built using HTML, CSS and JavaScript.

## 🌐 Live Website

[View Portfolio](https://sithumrd1-art.github.io/skillhub-portfolio/)

## 📌 About This Project

This portfolio website showcases my skills, projects, certificates, learning journey and contact information.

It is designed as a future-ready personal portfolio for job applications, LinkedIn profile, GitHub profile and interviews.

## ✨ Features

- Responsive design
- Dark mode
- Background image
- Project filter system
- Skill progress bars
- Typing animation
- Scroll animation
- CV download
- Certificate links
- Contact links
- GitHub Pages deployment
